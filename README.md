# WSL 문서
  
## 우분투 
### 버전 확인하기  
  
```
lsb_release -a
```
  
  
### 업그레이드 하기
  
```
sudo apt update
sudo apt upgrade
sudo apt dist-upgrade
sudo do-release-upgrade
```  
대부분 위 명령어를 실행하면 1시간 정도 걸린다.
